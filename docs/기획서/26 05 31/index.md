# **분위기**

- 경기 시작 전 긴장감 있는 느낌
- 어두운 실내에 위에서 떨어지는 스포트라이트
- 막 경기장으로 나가기 직전 같은 분위기
- “왕이 되기 전 대기하는 공간” 같은 느낌

# **배경에 들어가면 좋은 요소**

- **락커(사물함)**
- **벤치**
- **유니폼이나 수건**
- **농구공**
- **작전판 / 화이트보드**
- **바닥에 흩어진 프린트나 책**
- **청소도구나 학교 물건**
- 뒤쪽 문 너머로 경기장 조명이나 소리가 느껴지게 연출
