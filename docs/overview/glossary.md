# 용어집 (Glossary)

검색·링크의 기준이 되는 공통 용어. 새 용어는 여기에 먼저 등록한다.

| 용어 | 정의 | 관련 |
| --- | --- | --- |
| TTK | Time To Kill. 교전 시작~처치까지 시간(목표 3~5초). | [전투/데미지 시스템](../architecture/systems/combat-damage-system.md) |
| ESP | 벽 너머로 위치가 보이는 효과. 1등에게 주기적으로 발동. | [점수/랭크 시스템](../architecture/systems/scoring-rank-system.md) |
| 쉴드(Shield) | 체력 위에 덧대는 50 고정 보호막. 비전투 시 회복. | [전투/데미지 시스템](../architecture/systems/combat-damage-system.md) |
| 리벤지(Revenge) | 가장 최근에 나를 죽인 적을 되갚는 처치. 알림 발생. | [점수/랭크 시스템](../architecture/systems/scoring-rank-system.md) |
| 게이지(Gauge) | 청소기 등 특수 무기의 자원. | [무기 시스템](../architecture/systems/weapon-system.md) |
| RemoteEvent | 클라이언트↔서버 통신 객체. | [클라이언트-서버 모델](../architecture/client-server-model.md) |
