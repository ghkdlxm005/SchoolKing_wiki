# 게임 모드 — 1v1 중앙 거점 점령전

> 출처: 기획 노트 `26.06.21`. 초기 데스매치 컨셉([게임 컨셉 상세](./game-concept.md))에서 **1v1 거점 점령전**으로 구체화된 현재 방향이다.

## 1. 기본 방식

경기장 안에서 상대와 **1대1**로 싸우며, **중앙 거점에 머무는 동안에만** 점수가 오른다. 거점을 한 번 점령하고 끝이 아니라, 서 있는 내내 점수가 쌓이므로 플레이어가 계속 중앙으로 모이도록 유도한다.

## 2. 승리 조건

- 먼저 **목표 점수**에 도달한 플레이어 승리
- 예시: **목표 1000점 / 제한 5분**
- 5분 종료 시 목표 미달이면 **더 높은 점수**가 승리

## 3. 거점 점수 규칙

| 상황 | 점수 |
| --- | --- |
| 혼자 거점 안 | 증가 (예: 초당 5~10점) |
| 둘 다 거점 안 | **정지** |
| 거점 밖 | 증가 없음 |

거점 안에서 상대를 **처치하면 보너스**(예: 100~150점). 단순히 서 있는 게 아니라 중앙에서 적극 교전하도록 유도 — 리스크는 크지만 보상도 크다.

## 4. 스폰 / 리스폰

- 스폰 위치 = 맵의 **4개 모서리**, 중앙 근처에는 스폰하지 않음
- 시작 시 두 플레이어는 **서로 반대편 모서리**
- 리스폰 = 4개 모서리 중 **랜덤**, 패배 시 짧은 시간 후 즉시
- 상세 규칙·무적은 [스폰/리스폰 시스템](../architecture/systems/meta/spawn-respawn-system.md)

## 5. 파밍 시스템

맵 4개 모서리에 파밍 오브젝트 배치:

| 오브젝트 | 제공 |
| --- | --- |
| 자판기 | 캔음료 / 수류탄 |
| 책장 | 책 / 실드셀 |

파밍은 중앙에서 떨어진 곳에 있어 **선택**을 강요한다: 중앙에 남아 점수를 먹을지, 모서리로 빠져 회복·보급할지.

## 6. 회복 제한

회복 아이템은 쉽게 쓰지 못하도록 제한한다. 교전에서 이겨도 HP 손해가 크면 중앙에 남기 어렵고, 파밍하러 빠지면 그동안 상대가 점수를 얻는다. **핵심은 항상 중앙 거점 점수 획득**이 되도록 설계.

## 7. 전체 흐름

```text
경기 시작
→ 두 플레이어가 서로 반대편 모서리에서 스폰
→ 중앙 거점으로 이동
→ 혼자 거점 안 = 점수 획득 / 상대 진입 = 정지
→ 중앙 교전, 거점 안 처치 시 보너스
→ HP 부족하면 모서리 파밍 오브젝트로
→ 패배 시 4개 모서리 랜덤 리스폰
→ 먼저 1000점 도달 또는 5분 종료 시 승패 결정
```
