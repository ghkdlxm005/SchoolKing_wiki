# 프로젝트 개요 — School KING

Roblox 기반 **아레나 무브먼트 슈터**. 학교를 배경으로 일상 물건을 총기로 사용한다.

| 항목 | 값 |
| --- | --- |
| 장르 | 데스매치(포인트 기반) 무브먼트 FPS |
| 배경 | 학교 |
| 인원 | 최대 10명 (2v2 기반, 1v2도 성립) |
| TTK | 3~5초 |
| 핵심 재미 | "특색 있는 무기" + 빠른 무브먼트 |

## 이 위키의 역할

이 위키는 **코드보다 위에 있는 구조 레이어**다. 코드는 Claude가 생성하고,
구조 설계·검증·기록은 사람이 담당한다. 6개월 뒤에도 이 문서만으로 시스템을
복구할 수 있어야 한다.
