# 게임 컨셉 상세

## 한 줄 정의

> 일상 생활 속 물건을 총기로 사용하는, 학교 배경의 아레나 무브먼트 슈터.

## 핵심 루프

1. 랜덤 위치 스폰 → 2초 무적(공격은 가능)
2. 무브먼트로 교전, 무기로 상대 처치 → 포인트 획득
3. 사망 시 랜덤 리스폰, 가장 최근에 죽인 적이 나를 죽이면 "리벤지" 알림
4. 1등은 강조 효과(불타는 효과·왕관·주기적 ESP 노출)와 보너스를 받음

## 설계 의도

- **팀이 못해도 가능**: 무기 차별화와 무브먼트로 개인 기량이 빛나게.
- **특색 있는 무기**: 일상 물건을 무기로 재해석해 정체성을 만든다.
- 자세한 무기 목록은 [무기 시스템](../architecture/systems/weapon-system.md) 참고.
