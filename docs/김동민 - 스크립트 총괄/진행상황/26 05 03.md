# 26.05.03

### 작업 개요

기존 게임은 본인 시점(1인칭)에서만 무기와 발사 효과가 보이고, 다른 플레이어가 무엇을 들고 무엇을 하는지 시각적으로 확인할 수 없었음. 이번 작업으로 **멀티플레이** 환경에서 동료/적의 행동을 서로 인지할 수 있도록 3인칭 시각 표현 시스템을 구축함.

### 주요 작업 내용

**1. 무기 3인칭 표시**
모든 플레이어의 손에 들린 무기(컴퍼스, 컵, 캔)가 다른 플레이어 시점에서도 보이도록 함. 1인칭의 viewmodel과 완전히 동일한 형태는 아니지만, 같은 메쉬 자산을 사용해 무기를 식별할 수 있는 수준으로 손에 표시됨. (정확한 그립 위치/자세는 향후 미세 조정 필요)

**2. 시점 분리 처리**
본인 시점에서는 1인칭 viewmodel만 보이고 캐릭터 손의 3인칭 무기 모델은 숨김 처리. 무기가 두 번 겹쳐 보이는 시각적 혼란을 제거함.

**3. 발사 효과 동기화**
한 플레이어의 발사 동작이 모든 다른 플레이어 화면에 즉시 반영됨. 머즐플래시(총구 섬광), 트레이서(노란 총알 + 잔상), 3D 공간 발사음 세 가지가 동시에 재생되어 누가 어디서 쏘고 있는지 즉각 파악 가능.

**4. 수류탄 시각 일관성**
컵/캔 수류탄도 동일한 방식으로 처리. 들고 있을 때, 던질 때 모든 플레이어 시점에서 보임.

### 결과

멀티플레이 협업/대전이 가능한 시각 환경의 기반이 완성됨. 동료의 위치, 무기 종류, 행동이 시각적으로 전달되며, 향후 새로운 무기를 추가할 때도 같은 패턴(viewmodel 등록 → 효과 broadcast)으로 손쉽게 확장 가능한 구조를 갖춤.

### 알려진 한계 및 향후 과제

- 3인칭 무기의 정확한 그립 자세는 viewmodel과 100% 일치하지 않음 → Tool Grip Editor로 미세 조정 가능
- 캔(CAN) 3초 신관 카운트다운 시각/음향 표현
- 사망 사운드 보강
- 새로운 무기군(권총/샷건/소총) 추가
- PvP 점수/킬피드 UI, 인벤토리 시스템
