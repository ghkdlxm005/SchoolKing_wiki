# 스폰 / 리스폰 시스템

## 책임

최초 스폰, 사망 후 리스폰, 스폰 무적 처리.

## 확정 규칙

- 스폰 위치 **랜덤**
- 리스폰 **랜덤**
- 스폰 후 **2초 무적**, 단 무적 중에도 **공격 가능**
