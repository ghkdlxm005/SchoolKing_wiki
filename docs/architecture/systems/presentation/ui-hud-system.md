# UI / HUD 시스템

## 책임

체력/쉴드 바, 탄약·게이지, 킬로그, 1등/리벤지 알림, ESP 표시.

| 요소 | 소스 시스템 |
| --- | --- |
| 체력·쉴드 | [전투/데미지](../combat/combat-damage-system.md) |
| 탄약·게이지 | [탄약/게이지](../combat/ammo-system.md) |

## 크로스헤어 / 조준 (힙파이어, 구현됨 · 2026-06-30)

- **스카우터 스코프 제거** — 화면 중앙 **크로스헤어(힙파이어 에임)를 상시 표시**. 우클릭은 FOV 줌만(스코프 비주얼 없음).
- **다이내믹 크로스헤어** — `CrosshairController`가 장착 무기의 실제 탄퍼짐(base Spread × 이동상태 배율)에 맞춰 네 갈래 간격을 실시간 조절 + 발사 시 벌어졌다 회복. 탄착과 일치.
- 상세 → [FEAT-0006](../../../features/FEAT-0006-hipfire-accuracy.md)

## 크로스헤어 장탄 UI (기획 · `26.06.21`)

전투 중 **남은 장탄을 크로스헤어 주변에서** 직관적으로 보이게 한다(화면 구석 숫자 대신 주변시야로 파악). 조준 방해 안 되게 크로스헤어에서 **살짝 떨어진 위치**.

무기별 표시 예시:

| 무기 | 표시 |
| --- | --- |
| 토스터(더블배럴) | `■■` 2발 / `■□` 1발 / `□□` 장전 필요 |
| 실링폼/실리콘건 | 폼 게이지 `██████` → `███□□□` → `█□□□□□` |
| 컴퍼스 / 목발 | 작은 칸 여러 개 + 숫자 보조 |

> 상태: 기획. 구현 시 기존 [탄약/게이지](../combat/ammo-system.md) 어트리뷰트(`Ammo`/`AmmoReserve`)를 크로스헤어 UI에 바인딩.
