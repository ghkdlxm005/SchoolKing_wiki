# UI / HUD 시스템

## 책임

체력/쉴드 바, 탄약·게이지, 데미지 표기, 점수판, 적 외곽선 등 화면 표시 전반.

## 표시 항목

| 요소 | 소스 시스템 / 스크립트 |
| --- | --- |
| 체력·쉴드 | [전투/데미지](../combat/combat-damage-system.md) |
| 탄약·게이지 | [탄약/게이지](../combat/ammo-system.md) |
| 점수판·킬 피드백 | [점수/랭크](../meta/scoring-rank-system.md), `ScoreHUD` |
| 데미지 숫자·히트마커 | `DamageNumbers` (공유) |
| 상대 머리 위 쉴드 바 | `OverheadShield` |
| 적 외곽선 | `EnemyOutline` |

## 크로스헤어 / 조준

- **스카우터 스코프 제거** — 화면 중앙 크로스헤어 상시 표시. 우클릭은 FOV 줌만(스코프 비주얼 없음).
- **현재: 점(dot)만 표시** — 십자선(Top/Bottom/Left/Right)은 숨김, 중앙 `CenterDot`만 사용. (2026-07-12)
- `CrosshairController`의 탄퍼짐 연동 벌어짐 로직은 십자선용이라 화면상 효과 없음(코드는 유지).
- 힙파이어 정확도 배경 → [FEAT-0006](../../../features/FEAT-0006-hipfire-accuracy.md)

## 데미지 표기 (2026-07-12 개편)

- **단일 데미지**: 명중 지점 **좌측**, 부위별 색 — 머리 **노랑** / 몸통 **흰색** / 팔다리 **회색**. 동시 명중이 겹치지 않도록 좌우 지터.
- **합산**: **우측**에 같은 대상 0.7초 내 누적 합계 표시(점사·샷건·장판 틱이 하나로 합쳐짐).
- **투사체**(캔 폭발·컵 파편)도 숫자 표시, `AlwaysOnTop`이라 **벽 너머로도 수치 확인** 가능.
- 무기별 중복 표시 문제 → [BUG-0006](../../../bugs/BUG-0006-damage-number-duplicate.md)

## 점수판 (우상단)

등수(#1~#3 금·은·동)·프로필 얼굴·이름·점수, 본인 강조. 처치 시 `+N`과 **사유(흰 글씨)** 팝업.
→ [FEAT-0011](../../../features/FEAT-0011-score-hud-kill-feedback.md)

## 적 외곽선

상시 흰색 외곽선(`Occluded`, 투명도 0.5). **활성 연막**이 시선을 가리면 숨김 — 같은 연막 안에 둘 다 있으면 보임.
→ [FEAT-0009](../../../features/FEAT-0009-fire-extinguisher-smoke.md)

## 크로스헤어 장탄 UI (기획 · `26.06.21`)

전투 중 **남은 장탄을 크로스헤어 주변에서** 직관적으로 보이게 한다(화면 구석 숫자 대신 주변시야로 파악). 조준 방해 안 되게 크로스헤어에서 **살짝 떨어진 위치**.

무기별 표시 예시:

| 무기 | 표시 |
| --- | --- |
| 토스터(더블배럴) | `■■` 2발 / `■□` 1발 / `□□` 장전 필요 |
| 실링폼/실리콘건 | 폼 게이지 `██████` → `███□□□` → `█□□□□□` |
| 컴퍼스 / 목발 | 작은 칸 여러 개 + 숫자 보조 |

> 상태: 기획. 구현 시 기존 [탄약/게이지](../combat/ammo-system.md) 어트리뷰트(`Ammo`/`AmmoReserve`)를 크로스헤어 UI에 바인딩.
