# 웹 편집기로 페이지 만들기

마크다운을 몰라도 됩니다. **노션처럼 보이는 대로** 글을 쓰고, 새 페이지와 하위 페이지를 만들 수 있어요.

## 들어가기

1. 위키 오른쪽 위의 **✏️ 편집** 을 누르거나, 주소창에 직접:
   👉 **https://ghkdlxm005.github.io/SchoolKing_wiki/admin/**
2. **Login with GitHub** 버튼을 누르고 GitHub 계정으로 로그인합니다.
   (처음이라면 GitHub 무료 가입 후, 민지에게 아이디를 알려주면 편집 권한을 넣어줍니다.)

## 글 수정하기

1. 왼쪽 목록에서 고칠 페이지를 누릅니다.
2. 가운데 편집기에서 글을 고칩니다. 위쪽 버튼으로 **굵게, 제목, 목록, 이미지, 링크** 등을 넣을 수 있어요.
3. 오른쪽 위 **Publish → Publish now** 를 누르면 저장되고, 1~2분 뒤 위키에 반영됩니다.

## 새 페이지 만들기

1. 왼쪽 위 **New 위키 문서** (또는 + 버튼) 를 누릅니다.
2. **제목** 을 적고, **본문** 을 씁니다.
3. **위치(폴더 경로)** 칸에 어디에 둘지 적습니다.
   - 예: `기획서/새 아이디어` → 기획서 안에 "새 아이디어" 페이지가 생깁니다.
   - 슬래시(`/`)가 곧 상하위 관계입니다.
4. **Publish now**.

## 하위 페이지 만들기 (상하위 연결)

"위치(폴더 경로)" 를 한 단계 더 깊게 적으면 됩니다.

- `기획서` → 기획서 바로 아래
- `기획서/무기` → 기획서 > 무기
- `기획서/무기/청소기` → 기획서 > 무기 > 청소기

이렇게 슬래시로 단계를 늘리면 노션의 하위 페이지처럼 트리가 만들어집니다.

## 이미지 넣기

편집기 본문에서 **이미지 버튼**을 누르거나, 사진을 끌어다 놓으면 자동으로 올라갑니다.

## 순서 바꾸기

각 페이지의 **사이드바 순서** 숫자를 바꾸면 됩니다. 숫자가 작을수록 위에 표시됩니다.

:::tip
마크다운을 직접 쓰고 싶다면 [문서 수정하기(GitHub 방식)](../02-문서-수정하기/index.md) 도 있지만, 대부분은 이 웹 편집기만으로 충분합니다.
:::
