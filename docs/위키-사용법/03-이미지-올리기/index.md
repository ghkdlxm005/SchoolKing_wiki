# 이미지 · 그림 올리기

## 가장 쉬운 방법 (드래그 앤 드롭)

1. [2. 문서 수정하기](../02-%EB%AC%B8%EC%84%9C-%EC%88%98%EC%A0%95%ED%95%98%EA%B8%B0/index.md) 처럼 **Edit this page** 로 편집 화면을 엽니다.
2. 글 쓰는 칸에 이미지 파일을 **그대로 끌어다 놓습니다**.
3. GitHub이 이미지를 자동으로 올리고 `![](...)` 같은 링크를 넣어줍니다.
4. **Commit changes** 로 저장하면 끝.

:::tip
스크린샷은 복사(Ctrl+C) 후 편집칸에 붙여넣기(Ctrl+V) 해도 자동으로 올라갑니다.
:::

## 폴더에 직접 넣는 방법

이미지를 문서와 같은 폴더에 올린 뒤, 문서에서 이렇게 불러옵니다.

```markdown
![설명 글](./이미지이름.png)
```

- `![ ]` 안의 글자는 이미지가 안 보일 때 대신 나오는 설명입니다.
- 파일 이름에 한글·공백이 있어도 괜찮습니다.

## 주의할 점 

- **너무 큰 이미지는 피하세요.** 한 장에 2~3MB 이하면 충분합니다. 휴대폰 사진 원본은 용량이 크니 적당히 줄여서 올리는 게 좋습니다.
- 움직이는 그림(GIF)도 그대로 올릴 수 있습니다.
- 확장자(`.png`, `.jpg`)와 실제 파일 형식이 맞아야 합니다. (이름만 `.png` 인 JPG 파일은 깨져 보일 수 있어요.)
