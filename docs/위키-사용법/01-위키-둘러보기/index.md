# 위키 둘러보기 · 검색하기

## 위키 주소

👉 **https://ghkdlxm005.github.io/SchoolKing_wiki/**

이 주소를 즐겨찾기(북마크) 해두세요. 휴대폰에서도 그대로 열립니다.

## 문서 찾는 법

왼쪽 사이드바에서 카테고리를 눌러 문서를 펼쳐볼 수 있습니다.

- **기획서** — 날짜별 회의 기록
- **확정 내용 정리** — 확정된 게임 스펙
- **김동민 / 김준범 / 이준서** — 담당자별 작업 문서
- **테스트 에셋 · 참고자료** — 레퍼런스 모음

## 검색하기 🔍

화면 오른쪽 위의 **검색창**에 단어를 입력하면 위키 전체에서 찾아줍니다.

- 한국어로 검색됩니다. 예: `수류탄`, `리스폰`, `청소기`
- 인터넷 연결 없이도(오프라인) 검색이 동작합니다.
- 검색 결과를 누르면 해당 위치로 바로 이동하고, 찾은 단어가 노랗게 표시됩니다.

## 다크 모드 🌙

오른쪽 위 해/달 아이콘을 누르면 어두운 화면으로 바꿀 수 있습니다.
