## 문서 찾는 법

왼쪽 사이드바에서 카테고리를 눌러 문서를 펼쳐봅니다. 실제 구성은 이렇습니다.

- Project Overview — 게임 개념, 모드, 용어
- System Architecture — 시스템이 어떻게 굴러가는지 (코드 구조, 통신, 무기·전투·UI 등)
- Feature History — 기능별 개발 기록 (FEAT-XXXX)
- Bug History — 버그 이력 (BUG-XXXX). 증상 → 원인 → 조치 → 결과 순
- Lessons Learned — 버그에서 얻은 재발 방지 규칙 (LESSON-XXXX)
- TODO / Future Plan — 남은 작업, 로드맵
- 개발 일지 — 날짜별 개발 내역
- Keyword Index — 단어로 찾기 (Weapon, Ammo, UI, Event)
- Changelog — 굵직한 변경만 최신순으로

문서 ID 규칙: 기능은 `FEAT-####`, 버그는 `BUG-####`, 교훈은 `LESSON-####`. 서로 링크로 엮여 있어 하나를 열면 관련 문서로 넘어갈 수 있습니다.

## 검색하기 

화면 오른쪽 위의 **검색창**에 단어를 입력하면 위키 전체에서 찾아줍니다.

- 한국어로 검색됩니다. 예: `수류탄`, `리스폰`, `청소기`
- 인터넷 연결 없이도(오프라인) 검색이 동작합니다.
- 검색 결과를 누르면 해당 위치로 바로 이동하고, 찾은 단어가 노랗게 표시됩니다.
