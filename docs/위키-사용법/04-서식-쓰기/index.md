# 노션처럼 꾸미기 (서식 모음)

위키 문서는 **마크다운(Markdown)** 으로 씁니다. 노션의 `/` 슬래시 명령과 비슷하지만, 기호 몇 개만 외우면 됩니다. 아래를 그대로 따라 쓰세요.

## 제목

```markdown
# 큰 제목
## 중간 제목
### 작은 제목
```

## 글자 강조

```markdown
**굵게**  *기울임*  ~~취소선~~  `코드 글자`
```

→ **굵게**  *기울임*  ~~취소선~~  `코드 글자`

## 목록

```markdown
- 첫째
- 둘째
  - 안쪽 항목

1. 순서 1
2. 순서 2
```

## 체크리스트 

```markdown
- [x] 끝낸 일
- [ ] 할 일
```

- [x] 끝낸 일
- [ ] 할 일

## 인용 · 강조 박스

노션의 콜아웃처럼 눈에 띄는 박스를 만들 수 있습니다.

```markdown
:::tip 팁
유용한 정보를 여기에.
:::

:::warning 주의
조심할 내용을 여기에.
:::

:::note 메모
참고할 내용을 여기에.
:::
```
