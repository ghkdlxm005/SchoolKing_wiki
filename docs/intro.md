# School KING — 시스템 위키

Roblox 게임 **School KING**의 시스템 설계·기록 위키입니다.
이 위키는 **코드보다 위에 있는 구조 레이어**로, 6개월 뒤에도 시스템을 복구할 수 있게 모든 변경을 기록합니다.

## 이 위키 쓰는 법

버그를 만나면 Bug History에서 비슷한 사례를 찾고, 새 기능을 만들 땐 Feature History와 아키텍처 문서를 본다. 시스템이 어떻게 굴러가는지 궁금하면 Architecture, 폐기 없이 이력을 훑고 싶으면 Changelog와 개발 일지를 본다.
