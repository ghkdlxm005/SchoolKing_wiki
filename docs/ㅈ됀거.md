1.로블록스 관련 프로젝트 전부 잃어버림: 모델링,애니메이션

2.로블록스 플러그인 잃어버림 : 로블록스 내에서 사용 가능한 모델링 호환 프로그램들

5.마야 전용 플러그인도 사라짐 : 마야에서 쓰던 편의성 플러그인들

6.블렌더 파일도 사라짐 : 블렌더에 저장되어있던 로블록스 전용 import값이랑 호환값, 프로젝트 사라짐
