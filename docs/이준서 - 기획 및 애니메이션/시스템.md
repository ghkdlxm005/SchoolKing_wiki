# 시스템

## 게임 흐름

1. 두 명의 플레이어가 게임에 입장한다.
2. 양쪽 플레이어가 준비 완료 상태가 되면 라운드가 시작된다.
3. 각 플레이어는 맵의 양쪽 시작 위치에서 스폰된다.
4. 화면 중앙에 3, 2, 1 카운트다운이 표시된다.
5. START와 함께 UI가 표시되고 플레이어가 이동을 시작한다.
6. 플레이어는 교실 맵 안에서 이동과 무기를 활용해 교전한다.
7. 한 플레이어의 HP가 0이 되면 라운드 승패가 결정된다.
    - 쓰러지는 연출
    - 약간의 슬로우 효과
    - 점수판 갱신
    - 약 3초간 가한 데미지 / 받은 데미지 표시
8. 제한 시간이 끝났을 경우, 가한 데미지가 더 높은 플레이어가 승리한다.
    - 데미지가 같을 경우 남은 HP + 쉴드가 높은 플레이어가 승리한다.
9. 다음 라운드 시작 시 HP, 쉴드, 탄약, 위치가 초기화된다.
10. 결과 표시 후 다음 라운드로 넘어가며, 같은 흐름을 반복한다.
11. 먼저 3승 또는 5승을 달성한 플레이어가 최종 승리한다.
12. 최종 승리 시 승리 화면과 전체 전적이 표시된다.
