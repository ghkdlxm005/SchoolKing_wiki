# MVP (Minimum Viable Product)

전체적 흐름

움직임만 있는 플레이어> 사격 시스템 구현 > 게임 시스템 (루프) >맵구현, 맵 튜닝> 순위시스템 > 플레이 테스트

MVP완성 기준

- 한 판 더 하고 싶다
- 1등이 부담스럽다 (2, 3등이 쫓아오는 압박감, 긴장감)
- 잡았을 때 통쾌하다
- 죽어도 다시 뛰어들고 싶다
- 특정 구간에서 싸움이 계속 난다

최우선 순위 = 1번 최소한으로 ‘게임처럼 플레이 가능한 버전

먼저 개발하기 앞서, 주무기, 보조무기, 투척무기 에서 한 가지 씩 먼저 만들어보기 * 2월1일 정하기
